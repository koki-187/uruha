# uruha
特訓用学習管理システム

## 概要

勉強した内容を記憶し、ミニテストや本テスト対策として予習、復習、テスト出題、苦手科目克服ができる学習管理システムです。

## 機能

- 📚 **予習モード**: 学習内容を事前に確認
- 📝 **復習モード**: 学習履歴と統計を確認
- ✏️ **ミニテスト**: 短時間で復習できる小テスト
- 📋 **本テスト**: 総合的な理解度を確認する本格テスト
- 📉 **苦手科目分析**: 正答率から苦手科目を特定
- 💾 **学習データ管理**: 学習内容と問題を追加・保存

## 使い方

### 1. システムの起動

```bash
python3 study_system.py
```

### 2. サンプルデータの使用

サンプルデータを使って試す場合:

```bash
cp example_study_data.json study_data.json
python3 study_system.py
```

### 3. メニュー操作

起動後、以下のメニューから選択できます:

1. **学習内容を追加** - 新しい学習内容を登録
2. **問題を追加** - テスト用の問題を登録
3. **予習 (Preview)** - 学習内容を確認
4. **復習 (Review)** - 学習履歴を確認
5. **ミニテスト** - 短時間のテスト (問題数を選択可能)
6. **本テスト** - 全問題を使った総合テスト
7. **苦手科目分析** - 正答率から苦手科目を表示
8. **科目一覧** - 登録されている科目を表示
0. **終了** - システムを終了

## データ構造

### 学習内容 (StudyContent)

- **科目 (subject)**: 数学、英語、歴史など
- **トピック (topic)**: 具体的な学習項目
- **内容 (content)**: 学習内容の詳細
- **難易度 (difficulty)**: 1〜5の5段階評価
- **統計情報**: 復習回数、正答率など

### 問題 (Question)

- **科目 (subject)**: 問題の科目
- **問題文 (question)**: 問題の内容
- **答え (answer)**: 正解
- **選択肢 (options)**: 選択肢がある場合
- **難易度 (difficulty)**: 1〜5の5段階評価

## サンプルデータ

`example_study_data.json` には以下のサンプルが含まれています:

- **数学**: 二次方程式、三角関数
- **英語**: 過去完了形、関係代名詞
- **歴史**: 明治維新

各科目に学習内容と問題が用意されているので、すぐに試すことができます。

## 学習の流れ

1. **予習**: まず学習内容を確認して理解を深める
2. **ミニテスト**: 短時間のテストで理解度をチェック
3. **復習**: 学習内容を再確認し、記憶を定着させる
4. **本テスト**: 総合的な理解度を測定
5. **苦手科目分析**: 弱点を把握し、重点的に復習

## データの保存

- 学習データは `study_data.json` に自動保存されます
- 学習内容、問題、統計情報がすべて保存されます
- システムを再起動してもデータは保持されます
