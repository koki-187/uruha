# 06 Legal Pages Checklist

## 目的

My Agent Worksの会社HP、My Agent Series LP、宅建BOOST LPを公開・販売する前に必要な法務系ページと記載論点を整理する。

## 前提

- 本チェックリストは実装準備用であり、最終的な法務判断は専門家確認を前提とする。
- 宅建BOOSTは職能研修会との共同事業であり、販売主体・決済主体・問い合わせ窓口・運用保守主体を混同しない。
- 事業計画書では、共同LPは契約締結後に制作し、職能研様既存HPからリンク・バナーで誘導する方針が整理されている。

---

# 1. 必須ページ一覧

| Page | 必要度 | 対象 | 主な目的 |
|---|---|---|---|
| Privacy Policy | 必須 | HP / LP / フォーム | 個人情報取得・利用目的の明示 |
| Terms of Service | 必須 | My Agent Series / 宅建BOOST | 利用条件・禁止事項・免責 |
| Disclaimer | 必須 | AI / 投資 / 税務 / 法務 | AI出力の位置付け明示 |
| Specified Commercial Transaction Act | 決済開始時必須 | 有料販売ページ | 販売者・価格・返金条件 |
| Cookie Policy | 推奨 | GA4等導入時 | Cookie・分析ツール利用の明示 |
| Security / Trust | 推奨 | 会社HP | 信頼性・運用体制説明 |

---

# 2. プライバシーポリシー論点

## 取得する可能性がある情報

- 氏名
- メールアドレス
- 会社名
- 電話番号
- 問い合わせ内容
- 学習履歴
- 回答履歴
- 自己採点結果
- 実務経験年月
- コード利用状況
- 決済情報。ただしカード情報はStripe等の決済事業者が管理し、自社サーバーでは保持しない。

## 利用目的

- 問い合わせ対応
- サービス提供
- 本人確認
- 決済・請求
- 学習履歴・進捗の表示
- AI弱点分析
- 登録実務講習案内
- 法人ID管理
- サービス改善
- 不正利用防止
- 重要なお知らせの送信

## 第三者提供

```txt
初年度は原則として集計値共有を基本とする。個別ユーザーの実務経験年月、自己採点結果、講習関心情報を職能研様等へ共有する場合は、本人同意と契約上の取り決めを必須とする。
```

# 3. 利用規約論点

## 必須項目

- サービス概要
- 利用登録
- ID / パスワード管理
- 禁止事項
- 料金・支払方法
- 返金・キャンセル
- サービス停止・変更
- 知的財産権
- AI出力の扱い
- 免責事項
- 損害賠償
- 準拠法・管轄

## AI出力の扱い

```txt
本サービスにおけるAI出力は、学習、検索、整理、分析、資料作成を補助する情報であり、専門家判断、法的判断、投資判断、税務判断、合格保証を行うものではありません。
```

# 4. 特定商取引法表記論点

## 必須項目

- 販売事業者名
- 運営責任者
- 所在地
- 電話番号または問い合わせ方法
- メールアドレス
- 販売価格
- 商品代金以外の必要料金
- 支払方法
- 支払時期
- 提供時期
- 返品・キャンセル・返金条件
- 動作環境

## 宅建BOOSTの注意

宅建BOOSTは職能研修会との共同事業であるため、以下を確定する必要がある。

| 項目 | 要確認 |
|---|---|
| 販売主体 | 職能研修会か、My Agent Worksか |
| Stripe名義 | 職能研修会名義か、My Agent Works名義か |
| 入金口座 | 職能研修会名義か、My Agent Works名義か |
| 問い合わせ一次窓口 | 職能研名義か、My Agent Works名義か |
| 実務対応 | My Agent Worksが担うか |
