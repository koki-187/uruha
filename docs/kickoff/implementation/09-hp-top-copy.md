# 09 HP Top Copy

## 目的

合同会社 My Agent Works 公式HPトップページに掲載する実装用コピーを確定する。  
本コピーは、`08-hp-top-wireframe.md` と `08a-hp-design-inheritance-rules.md` に基づき、信頼性・先進性・実務性を両立する。

## コピー設計方針

- 会社HPは、商品を強く売り込む前に法人としての信頼を作る。
- HeroとFinal CTAではCinematic PropTechの世界観を出す。
- 本文セクションはTrust Monotoneを中心に、金融機関・税理士・司法書士・協業先にも見せられる品位を保つ。
- 不動産仲介業者と誤認される表現を避ける。
- AIが投資・税務・法務・登録判断を代替するような表現は使わない。
