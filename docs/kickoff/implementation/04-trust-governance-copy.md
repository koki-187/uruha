# 04 Trust / Governance Copy

## 目的

AI・不動産・税務・投資判断が絡むサービスとして、利用者・協業先・金融機関が安心して確認できる信頼ページの本文を準備する。

## ページタイトル案

```txt
Trust / Governance
信頼とガバナンス
```

## リードコピー案

```txt
My Agent Worksは、AIを業務判断の代替ではなく、実務を支援するための補助ツールとして設計しています。情報の取り扱い、出力結果の確認、専門家判断が必要な領域を明確にし、安心して利用・相談できる運用体制を整備します。
```

## 1. AI出力の位置付け

```txt
AIによる出力は、検索・整理・分析・資料作成を効率化するための補助情報です。最終的な取引判断、投資判断、税務判断、法務判断を代替するものではありません。
```

## 2. 不動産・投資判断について

```txt
不動産取引や投資判断では、物件の個別事情、市場環境、法令、税務、融資条件など複数の要素を総合的に確認する必要があります。My Agent Worksの各プロダクトは、判断に必要な情報整理を支援しますが、最終判断は利用者または専門家による確認を前提とします。
```

## 3. 税務・法務判断について

```txt
税務・法務に関する表示や出力は、一般的な情報整理を目的としたものであり、税理士、弁護士、司法書士その他専門家の判断を代替するものではありません。必要に応じて、各分野の専門家へ確認してください。
```

## 4. 情報管理について

```txt
顧客情報、物件資料、契約関連資料などの機密性が高い情報を扱う場合、利用目的・保存範囲・共有範囲を明確にし、必要最小限の利用を前提に運用します。
```

## 5. 外部AI/APIの利用について

```txt
一部機能では、AI API、OCR、クラウドストレージ、決済サービス等の外部サービスを利用する場合があります。利用する外部サービスの範囲は、導入時または利用規約等で明示します。
```

## 6. 共同事業の運用主体について

```txt
宅建BOOSTなど、協業先と共同で提供するサービスについては、販売主体、決済主体、問い合わせ窓口、運用保守の役割を整理し、利用者に誤解が生じないよう明示します。
```

## 7. 保守・問い合わせ対応について

```txt
サービス導入後の問い合わせ、エラー報告、改善要望については、提供範囲に応じて対応します。個別開発・カスタマイズ・追加機能については、別途見積または契約条件に基づき対応します。
```

## FAQ案

### Q. AIの出力だけで不動産投資判断をしてよいですか？

```txt
いいえ。AI出力は判断材料の整理を支援するものであり、最終的な投資判断を保証するものではありません。
```

### Q. 税務や法務の相談もできますか？

```txt
一般的な情報整理や資料作成の支援は可能ですが、税務・法務の最終判断は税理士・弁護士・司法書士等の専門家に確認してください。
```

### Q. 顧客資料をAIに入力しても大丈夫ですか？

```txt
導入時に、利用するAI/API、保存範囲、取り扱うデータ種別を確認し、必要最小限の情報利用を前提に設計します。
```

## 実装時の注意

- 不安を煽らず、安心材料として表現する。
- 過度な法的断定は避ける。
- 利用規約・プライバシーポリシーと整合させる。
