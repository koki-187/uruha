# 02 Product Sales Status

## 目的

## ステータス定義

| ステータス | 意味 | HP/LPでの表現 |
|---|---|---|
| 販売可能 | すぐ提供できる | 提供中 / 導入相談受付中 |
| β提供可能 | 限定的に提供できる | β提供 / パイロット導入相談 |
| デモ可能 | 画面や資料で説明できる | デモ相談受付中 |
| 構想段階 | まだ販売しない | Coming Soon / 開発構想 |
