# Cinematic PropTech Design System

## 1. 目的

My Agent Worksを、単なるAIツール販売会社ではなく、**不動産実務をAIで再設計するPropTechカンパニー**として印象づける。

## 2. 使用シーン

- 会社HPのHero
- My Agent Series LP
- My Agent Series営業資料表紙
- 事業ビジョンページ
- CTAセクション
- プロダクト紹介セクション
- 16:9告知画像
- Agent Command Center構想ページ

## 3. 使用を控えるシーン

- 会社概要
- 法人情報
- 金融機関向け資料
- 税理士向け資料
- 司法書士向け資料
- プライバシーポリシー
- 利用規約
- Trust / Governance本文

## 5. 採用する表現

- 夜の都市 skyline
- 高層ビル群
- 光のグラデーション
- 薄いフィルムグレイン
- グリッドオーバーレイ
- HUD風の細線
- AIノード
- シールド型プロダクトチップ
- 線画アイコン
- ガラスモーフィズムカード
- 左側にコピー、右側にビジュアルの構成
- 黒に近いネイビー背景にCyanとGoldを少量使う

## 6. 避ける表現

- サイバーパンクに寄せすぎる
- 紫・ピンク・赤の多用
- ゲームUI感
- 過度なネオン
- 意味のない3Dオブジェクト
- テキストが読みにくい暗さ
- 全ページ黒背景
- 情報商材風の強い煽り
- AIが何でも自動で解決するような誇張表現

## 7. 推奨画像カテゴリ

| カテゴリ | 用途 |
|---|---|
| HP TOP HERO | 会社HPトップ |
| HP BUSINESS DOMAIN | 事業領域 |
| HP TRUST GOVERNANCE | 信頼・ガバナンス |
| LP HERO | My Agent Series LP |
| LP WORKFLOW MAP | 業務フロー |
| LP COMMAND CENTER | 統合管理画面 |
| LP TAKKEN HERO | 宅建BOOST |
| COMMON ASSET | アイコン・チップ・UIパーツ |

## 8. 最終方針

会社HPは信頼性を守るため限定的にCinematic PropTechを採用し、My Agent Series LPでは最大限に活かす。宅建BOOST LPはCyan中心の教育DX表現へ調整する。
