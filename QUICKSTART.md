# クイックスタートガイド

## 最速で始める方法

### ステップ1: サンプルデータをコピー

```bash
cp example_study_data.json study_data.json
```

### ステップ2: システムを起動

```bash
python3 study_system.py
```

### ステップ3: メニューから選択

起動したら、以下のような操作を試してみてください:

## 基本的な使い方の例

### 1. 科目一覧を見る
```
選択: 8
```

### 2. 予習する（数学の内容を確認）
```
選択: 3
科目: 数学
```

### 3. ミニテストを受ける（5問）
```
選択: 5
科目: 数学
問題数: 5
```

### 4. 苦手科目を確認
```
選択: 7
```

### 5. 復習する
```
選択: 4
科目: (空白で全科目)
```

## デモを見る

全機能のデモを見るには:

```bash
python3 demo.py
```

## テストを実行

システムが正しく動作するか確認:

```bash
python3 test_study_system.py
```

## 新しい学習内容を追加

1. メニューで「1」を選択
2. 科目を入力（例: 数学）
3. トピックを入力（例: 微分）
4. 内容を入力（例: f'(x) = lim...）
5. 難易度を入力（1-5）

## 新しい問題を追加

1. メニューで「2」を選択
2. 科目を入力
3. 問題文を入力
4. 答えを入力
5. 選択肢を追加するか選択（y/n）
6. 難易度を入力（1-5）

## ヒント

- 📚 **予習**: テストを受ける前に内容を確認
- ✏️ **ミニテスト**: 短時間で理解度をチェック
- 📋 **本テスト**: 全問題で総合力を測定
- 📝 **復習**: 学習履歴と正答率を確認
- 📉 **苦手科目分析**: 弱点を特定して重点学習

## データの保存

- すべてのデータは `study_data.json` に自動保存されます
- システムを終了しても、次回起動時に自動的に読み込まれます
- バックアップを取る場合は、このファイルをコピーしてください

## トラブルシューティング

### エラーが出る場合

Python 3.6以上がインストールされているか確認:
```bash
python3 --version
```

### データをリセットしたい場合

```bash
rm study_data.json
cp example_study_data.json study_data.json
```

### まっさらな状態から始めたい場合

```bash
rm study_data.json
python3 study_system.py
```
